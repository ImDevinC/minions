# Progress Log
PRD: minions-system
Started: 2026-03-21

## Codebase Patterns
- Migrations: `schema/migrations/NNN_description.sql` - raw SQL, no ORM
- UUID PKs with `gen_random_uuid()` default
- Timestamps: `TIMESTAMPTZ NOT NULL DEFAULT NOW()` for created_at/updated_at
- Go services: chi router, JSON handlers, slog for structured logging
- Router setup in `internal/api/router.go`, entry point in `cmd/<service>/main.go`
- Interface stubs: Use interface + NoOp implementation for external dependencies (k8s, webhooks) to allow incremental implementation
- Handler config structs: Use `FooHandlerConfig` struct for >3 dependencies instead of positional params
